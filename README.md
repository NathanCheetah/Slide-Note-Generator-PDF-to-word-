Unlimited PDF Files

Process:
Pushes PDF together to give to claude

Output:
Markdown file



HOW TO RUN LOCALLY:
Go to your GitHub repo → click the green "Code" button → "Download ZIP"
Unzip it, open a terminal in that folder and run:
  npm install
  npm run dev

Open http://localhost:5173 in your browser
You'll need Node.js installed first — get it at nodejs.org (the LTS version).
