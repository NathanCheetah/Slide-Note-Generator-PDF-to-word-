# Slide Notes Generator

Upload PDF slide decks and get AI-structured notes organised by topic and subtopic.

## Setup

```bash
npm install
npm run dev
```

## Usage

1. Upload one or more PDF slide decks
2. Choose your detail level
3. Click Generate — Claude AI organises the content into structured notes
4. Download as Markdown or copy to clipboard

## Deploy

Connect this repo to [Vercel](https://vercel.com) for a free live URL.
